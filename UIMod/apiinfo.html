REST API Information

This server is based on Go, so it's basically a REST-API with some html files on top. All UI Actions are API calles, so you can fully use the API to control the server.

  • /start GET
  • /stop GET
  • /output GET
  • /backups GET
  • /restore GET with index parameter: /restore?index=123
  • /saveconfig POST Form Data, see below
  • /config GET

Form Data Explanation

SaveFileName: The name of the save file to load. This is the name of the file without the extension. Example: Mars

Settings: The server settings. If you need the API, reverse engeneer the form yourself. Actually, take this as a test: If you are not able to do that, you probably should just use the UI instaed of the API ^^
